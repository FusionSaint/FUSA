## üîÑ Rollback Procedures

<div class="space-y-6 mb-8">

</div>

<div class="grid grid-cols-1 md:grid-cols-2 gap-6 mb-8">
  <div class="p-6 bg-card/50 rounded-lg border-l-4 border-primary">

    <ul class="text-sm space-y-2 text-foreground/80">
      <li>‚úì Dying to recoil? ‚Üí Upgrade grip (Folding ‚Üí Ribbed ‚Üí Alloy ‚Üí Classic)</li>
      <li>‚úì Dying off-minimap? ‚Üí Add Standard Suppressor (25pts)</li>
      <li>‚úì Losing CQB? ‚Üí Add Blue Laser (30pts) or Short Barrel (10-15pts)</li>
      <li>‚úì Losing long-range? ‚Üí Add Extended Barrel (25pts) + upgrade optic</li>
    </ul>
  </div>

  <div class="p-6 bg-card/50 rounded-lg border-l-4 border-green-500">
    <div class="text-xs text-green-400 font-semibold mb-3">STEP 2: CHECK POINT BUDGET</div>
    <ul class="text-sm space-y-2 text-foreground/80">
      <li>‚úì &lt;30pts remaining ‚Üí Build is optimized, issue is mechanical skill</li>
      <li>‚úì 30-50pts remaining ‚Üí Under-invested, add core attachments</li>
      <li>‚úì 50-100pts remaining ‚Üí No build identity, restart from scratch</li>
    </ul>
  </div>

  <div class="p-6 bg-card/50 rounded-lg border-l-4 border-yellow-500">
    <div class="text-xs text-yellow-400 font-semibold mb-3">STEP 3: A/B TEST SINGLE VARIABLE</div>
    <ul class="text-sm space-y-2 text-foreground/80">
      <li>‚úì Change ONE attachment per test session (5-10 matches)</li>
      <li>‚úì Track K/D, average damage, engagement range deaths</li>
      <li>‚úì Revert if performance drops &gt;10%</li>
    </ul>
  </div>

  <div class="p-6 bg-card/50 rounded-lg border-l-4 border-fs-gold">
    <div class="text-xs text-fs-gold font-semibold mb-3">STEP 4: META BUILD COMPARISON</div>
    <ul class="text-sm space-y-2 text-foreground/80">

      <li>‚úì Play 10 matches with meta build</li>

    </ul>
  </div>
</div>
