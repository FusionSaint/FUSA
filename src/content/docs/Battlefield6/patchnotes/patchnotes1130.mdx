### CPU Optimization - user.cfg Method

<div class="p-6 bg-card/30 rounded-lg border border-border mb-8">
<div class="text-sm font-semibold text-foreground mb-4">Manual CPU Thread Configuration</div>
<div class="text-xs text-foreground/80 mb-4">
Some players report significant performance improvements by manually configuring how Battlefield 6 utilizes CPU cores. This fix addresses Windows misclassifying the game as a low-priority process and throttling performance.
</div>

<div class="space-y-4">
<div class="p-4 bg-card/50 rounded">
<div class="text-xs font-semibold text-primary mb-2">STEP 1: Find Your CPU Core/Thread Count</div>
<div class="text-xs text-foreground/80 space-y-1">
<p>‚Ä¢ Press <kbd>Ctrl + Shift + Esc</kbd> to open Task Manager</p>
<p>‚Ä¢ Go to Performance tab ‚Üí CPU</p>
<p>‚Ä¢ Note <strong>Cores</strong> (e.g., 8) and <strong>Logical processors</strong> (e.g., 16)</p>
</div>
</div>

<div class="p-4 bg-card/50 rounded">
<div class="text-xs font-semibold text-primary mb-2">STEP 2: Create user.cfg File</div>
<div class="text-xs text-foreground/80 space-y-1">
<p>‚Ä¢ Navigate to: <code>C:\Program Files (x86)\Steam\steamapps\common\Glacier Events</code></p>
<p>‚Ä¢ Create new text file named: <code>user.cfg</code></p>
<p>‚Ä¢ Open with Notepad</p>
</div>
</div>

<div class="p-4 bg-card/50 rounded">
<div class="text-xs font-semibold text-primary mb-2">STEP 3: Add Configuration</div>
<div class="text-xs text-foreground/80 mb-2">Copy this template and replace values with YOUR numbers:</div>
<pre class="text-xs bg-card/70 p-3 rounded overflow-x-auto"><code>Thread.ProcessorCount [YOUR CORE COUNT]
Thread.MaxProcessorCount [YOUR CORE COUNT]
Thread.MinFreeProcessorCount 0
Thread.JobThreadPriority 0
Thread.MaxWorkerThreadCount [YOUR THREAD COUNT]</code></pre>
<div class="text-xs text-fs-gold mt-2">Example for 8-core/16-thread CPU:</div>
<pre class="text-xs bg-primary/10 p-3 rounded overflow-x-auto mt-1"><code>Thread.ProcessorCount 8
Thread.MaxProcessorCount 8
Thread.MinFreeProcessorCount 0
Thread.JobThreadPriority 0
Thread.MaxWorkerThreadCount 16</code></pre>
</div>

<div class="p-4 bg-card/50 rounded">
<div class="text-xs font-semibold text-primary mb-2">STEP 4: Save and Lock File</div>
<div class="text-xs text-foreground/80 space-y-1">
<p>‚Ä¢ Save the file</p>
<p>‚Ä¢ Right-click ‚Üí Properties ‚Üí Check "Read-only"</p>
<p>‚Ä¢ This prevents the game from overwriting your settings</p>
</div>
</div>
</div>

<strong>Reported Results:</strong> CPU usage in menu drops from 35% to 20%, in-game drops from 70-80% to 30-40%, with significantly reduced stuttering.
