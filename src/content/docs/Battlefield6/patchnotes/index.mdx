## Coverage Schedule

<div class="space-y-4 mb-8">

**We create major patch documentation for:**
- ✅ Seasonal updates with new content
- ✅ Significant weapon rebalancing affecting meta
- ✅ Major game mode or map changes
- ✅ Core mechanics overhauls (movement, netcode, etc.)

**We don't track:**
- ❌ Backend stability patches
- ❌ Minor hotfixes without gameplay impact
- ❌ Server-side tweaks that don't shift competitive strategy
